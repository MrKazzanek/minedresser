# MineDresser

**MineDresser** is a web-based tool that works like a wardrobe for Minecraft Bedrock players. It allows users to load their Minecraft skin and dress it with custom outfits available on the site.

👉 [https://minedresser.netlify.app/](https://minedresser.netlify.app/)

## ✨ Features

* Load your Minecraft skin using your nickname (fetched from Mojang servers)
* Upload your own skin (`.png`, 64x64 or 64x32)
* Dress your character with available clothing items
* Supports both **Steve (wide)** and **Alex (slim)** models
* Fully browser-based – no login required

## 🚀 How It Works

1. Open the website
2. Load your skin:

   * Enter your Minecraft nickname
   * or upload your own `.png` skin file
3. Browse available clothing
4. Customize your character in real time

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Font Awesome
* SkinView3D

## 🌐 Usage

MineDresser runs entirely in the browser. No installation needed.

Just open:
👉 [https://minedresser.netlify.app/](https://minedresser.netlify.app/)

Make sure your browser supports **SkinView3D**.

## 👥 Community & Contributions

The project is developed by **QozaWorks**, but the community can contribute by submitting their own clothing.

### 📦 Clothing Submission Requirements

To submit a clothing item, you must provide:

* Clothing texture
* File format: `.png`
* Texture size: **64x64px** or **64x32px**
* File name = clothing name
* Category (e.g. Hats — can be a new category)
* Description
* Author name
* Social media link (optional)
* Skin type:

  * Steve (wide)
  * Alex (slim)
  * All

### 📜 Rules

* You must own the rights to the submitted clothing
* No NSFW content allowed
